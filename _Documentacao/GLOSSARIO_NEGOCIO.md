## 📂 Contextos de Visibilidade Financeira

### 1. Esteira Financeira (Grid de Detalhamento / Lista)
Estas siglas são usadas nas colunas do grid principal para mostrar a evolução do fluxo financeiro.

| Sigla | Significado |
| :--- | :--- |
| **RE** | **Realizado** |
| **EM** | **Empenhado** |
| **CO** | **Comprometido** |
| **ES** | **Estimado** |
| **NC** | **Não Comprometido** |

### 2. Outlook da Obra (Cards de Topo / Dashboard)
Estas siglas são usadas nos indicadores consolidados no topo do Dashboard.

| Sigla | Significado |
| :--- | :--- |
| **NC** | **Não Comprometido** |
| **CP** | **Comprometido** |
| **EM** | **Empenhado** |
| **CT** | **Contabilizado** |
| **MC** | **Mobilização Contratual** |
| **PP** | **Parada Programada** |
